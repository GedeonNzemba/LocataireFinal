How to remove a watermark on the created slider:
1) Go to the folder where you saved the slider. 
2) The data \ images folder contains a video or image on which is superimposed a watermark. 
Take the same picture or video to create a slider and replace them. 
3) Because of the distribution of take wowslider.js and we place a file in a folder on engine replacement.
4) Open the index.html file in a web browser and enjoy slider without watermark.
